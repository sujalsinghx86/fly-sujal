# FLIGHT API

ENDPOINT <br>
https://fly-sujal.herokuapp.com/?oci={}&oco={}&dci={}&dco={}&dd={}

### PARAMETERS

1. oci
    - `ORIGIN CITY`
    - `REQUIRED`
2. oco
    - `ORIGIN COUNTRY`
    - `REQUIRED`
3. dci
    - `DESTINATION CITY`
    - `REQUIRED`
4. dco
    - `DESTINATION COUNTRY`
    - `REQUIRED`
5. dd
    - `DEPARTURE DATE`
    - `REQUIRED`
    - `FORMAT: 16 august 2021`
